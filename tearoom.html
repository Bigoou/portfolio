7ea#Room

Novembre 2021

7ea#Room est le résultat d'un projet de classe se nommant hexword. Le concept était de choisir un code hexadécimal, qui pouvait être interprété comme un mot, et de faire un site autour de cette thématique en HTML / CSS. J'ai choisi comme code #7ea (ou #7EEAA), qui peut aussi être lu comme Tea. Mon site présente une boîte de thé contenant des sachets, qui donnent des informatons sur différents types de thé (bienfaits, comment le consommer...).

SKILLS

HTML, CSS, Javascript Vanilla, Figma
